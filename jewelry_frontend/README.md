# Jewelry Shop 💎

Full Stack E-commerce Jewelry Application

## Tech Stack

Frontend:
- React JS
- React Router
- CSS

Backend:
- Django
- Django REST Framework
- JWT Authentication

Database:
- SQLite


## Features

- User Registration/Login
- JWT Authentication
- View Jewelry Products
- Search Products
- Add to Cart
- Checkout
- Place Orders
- View Order History
- Cancel Orders

Admin Features:

- Add Products
- Update Products
- Delete Products
- View Customer Orders
- Update Order Status


## Run Frontend

npm install

npm run dev


## Run Backend

pip install -r requirements.txt

python manage.py runserver
