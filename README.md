# Jewelry Ecommerce Website 💎

A full stack ecommerce application built using React and Django REST Framework.

## Features

- User Authentication (JWT)
- Product Management
- Admin Dashboard
- Add/Edit/Delete Products
- Shopping Cart
- Checkout
- Order Management
- Order Status Update
- Image Upload

## Technologies

Frontend:
- React.js
- React Router
- CSS

Backend:
- Django
- Django REST Framework
- JWT Authentication

Database:
- SQLite
